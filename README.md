# 数据目录

此目录用于存放数据文件。

## 说明

本项目不包含原始数据文件，你需要准备自己的数据。

## 数据格式

Excel 文件应包含以下列：

| 列名 | 说明 | 类型 |
|-----|------|------|
| 结算编码 | 项目编码 | 字符串 |
| 结算客户 | 客户名称 | 字符串 |
| 省区平台 | 省份 | 字符串 |
| 城市 | 城市 | 字符串 |
| 行业 | 行业分类 | 字符串 |
| 项目简称 | 项目名称 | 字符串 |
| 日期 | 报表日期 | 日期 |
| 业绩量 | 业绩指标 | 数值 |
| 销售收入 | 收入 | 数值 |
| 经营利润 | 利润 | 数值 |

## 数据导入

将数据文件放在此目录，然后运行：

```bash
python import_xlsx.py
```

## 注意

此目录下的文件（.xlsx, .csv）不会被提交到 Git。
